# 🦊 Fox developer Toolkit

Welcome to my bite-sized collection of developer discoveries!

This repository is where I collect the libraries, tools, databases, and other technologies that have earned a permanent place in my developer toolbox.

Rather than documenting every feature, each folder contains the smallest practical example I could build—just enough to understand what the technology does, when I'd use it, and why I think it's worth keeping around.

Feel free to look around, take what you need and also [leave some recommendations](https://github.com/FanniBelics/Fox_Developer_Toolkit/issues) if you have any!

I also share these discoveries on my [LinkedIn profile](https://www.linkedin.com/in/fanni-belics-545a8a296/), where I talk about how I use them in real projects and why they earned a place in my toolbox. 🔗
