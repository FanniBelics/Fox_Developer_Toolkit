## Recommendation
Not to overflow your Python library with all the tests and new modules I recommend you to create a test environment, where you can manage the installed modules easily!
You have never made a virtual environment? Take this easy walkthrough:
1. Clone the this repository anywhere on your computer
`git clone https://github.com/FanniBelics/Fox_Developer_Toolkit.git`
2. Open your command prompt (cmd) in the directory
3. Create a virtual environment using [conda](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html)
`conda create --name test_env python=3.14`
4. Activate your environment
`conda activate test_env`
